==How to Run==

1. Locate the directory that has hw4compiler.c file

2. In command line type "gcc hw4compiler.c -o main"
- This will compile the code

3. In command line type "./main (enter text file here)
- This will run Compiler using instructions from the given text file.


Output will be displayed in Output File.
